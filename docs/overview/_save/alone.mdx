## 1. DTO Code

```sh
export yourpackage.Book
    -> yourpackage.dto

input BookInput {
    #allScalars
}
```

After compilation, Java/Kotlin type `BookInput` will be automatically generated.

## 2. Code for saving

<Tabs groupId="language">
<TabItem value="java" label="Java">

```java
@PutMapping("/book")
public int saveBook(
    @RequestBody BookInput input
) {
    return sqlClient
        .save(input)

}
```
